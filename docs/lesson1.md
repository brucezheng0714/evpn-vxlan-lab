# 第一课：为什么我开始从 EVE-NG 转向 Containerlab

## EVE-NG 的优点

EVE-NG 作为传统的网络虚拟化实验平台，拥有成熟的 GUI 界面和大量社区资源。

---

## EVE-NG 的问题

随着 EVPN/VXLAN 实验规模越来越大，传统 VM 实验逐渐暴露出：

- 内存占用高
- 启动速度慢
- 节点扩展困难
- 自动化能力有限
